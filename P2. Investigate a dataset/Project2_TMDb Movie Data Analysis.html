Introduction¶

This dataset was taken from The Movie Database (TMDb). "a community built movie and TV database." (www.themoviedb.org/about) Each row corresnponds to a movie and includes a range of data about each film. Relevant data to be used in the following analysis includes the following varialbes:

  • genre
  • popularity
  • budget
  • revenue
  • runtime
  • vote
  • release_year

In this report, I explore the following questions:

  1. Which year has the highest movie release?
  2. Which movie genre is the most profitable?
  3. Which movies are top 10 profitable movies?
